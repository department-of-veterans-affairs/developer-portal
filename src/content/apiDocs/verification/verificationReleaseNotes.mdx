# Address Validation

## September 26, 2018
Create Address Validation API [#336](https://github.com/department-of-veterans-affairs/vets-contrib/commit/646a1c45df3c58f361ea523afc9e772dfa07999a)
- The ‘validate’ API accepts a structured JSON address object (broken down by street/city/state/zip/etc), and returns a result indicating whether the address is valid, and if so includes a canonicalized address and geocoding information (lat/long) for the address.

# Veteran Verification

## November 8, 2018
Launched the Disability Rating and Veteran Status endpoints
- Disability Rating endpoint for external API consumers. Retrieves disability rating of an authorized Veteran.
- Allows third-parties to request confirmation from the VA of an individual's Veteran status after receiving authorization to do so using an Open ID Connect flow.

## June 11, 2018 
Expose Service History endpoint via API gateway as POC [#172](https://github.com/department-of-veterans-affairs/vets-contrib/issues/172)
- Given an authenticated user, this created an API endpoint that returns their service history resolved from eMIS
