### June 23, 2022

(v1) We added a new end point to the VA Letter Generator API which returns JSON formatted text versions of letters the Veteran is eligible to receive.

Learn more about this API by reviewing the VA Letter Generator Documentation.

For cross-reference purposes, the PR for this release is [#83](https://github.com/department-of-veterans-affairs/lighthouse-va-letter-generator/pull/83)
