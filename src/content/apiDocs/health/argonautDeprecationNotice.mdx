## Deprecation Notice
Consumers currently using the <code>https://api.va.gov/services/argonaut/v0</code> (and <code>https://dev-api.va.gov/services/argonaut/v0</code>) endpoint, please migrate your applications to use the new Veteran Health API (FHIR) endpoints: <code>https://api.va.gov/services/fhir/v0/argonaut/data-query/</code> and <code>https://dev-api.va.gov/services/fhir/v0/argonaut/data-query/</code>

Both the legacy API endpoints and this legacy documentation page will no longer be accessible beginning Oct 1, 2019
